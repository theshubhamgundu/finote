## About the Project

**Finote** ‚Äì Master Your Finances Take control of your financial life with Finote, a powerful and intuitive expense tracker built with React Native. Effortlessly manage multiple wallets, track your daily income and expenses, and visualize your spending habits with beautiful, real-time statistics. Whether you're budgeting for a goal or just tracking cash flow, Finote makes personal finance simple and secure.

## üî• Key Features

- **üí∞ Multi-Wallet Management**
  Create and manage multiple wallets (e.g., Salary, Freelancing, Cash) to keep your finances organized.

- **üìä Visual Analytics**
  Detailed weekly, monthly, and yearly bar charts to visualize your income vs. expense trends.

- **üìù Smart Transaction Tracking**
  Easily log income and expenses with categories, dates, and descriptions.

- **üßæ Receipt Uploads**
  Attach images/receipts to your transactions (powered by Cloudinary) so you never lose a bill.

- **üîç Advanced Search**
  Quickly find specific transactions by searching for keywords, categories, or transaction types.

- **üîê Secure Authentication**
  Complete sign-up and login system powered by Firebase Auth.

- **üé® Smooth UI/UX**
  Features a custom tab bar, engaging animations, and a sleek responsive design.

## üß∞ Setup

1. Make sure [Git](https://git-scm.com/downloads) and [Node.js](https://nodejs.org/en/) are installed.
2. Fork this repository on GitHub and clone **your fork** to your local machine:

   ```bash
   git clone https://github.com/<your-username>/finote-app.git
   cd finote-app
   ```
3. Open the terminal in the project root and run `pnpm install` to install dependencies.
4. Start the development server with `pnpm dev`.
